ℍ - Quatérnios

Os quatérnios são uma extensão ℍ do conjunto dos números complexos ℂ. Uma das principais aplicações dos quatérnios é a rotação de vetores em 3D. Eles também possuem aplicações na computação gráfica, na robótica e em qualquer coisa que envolva realidade virtual.

Three.js

A biblioteca Three.js permite a criação de animações 3D aceleradas por GPU (Unidade de Processamento Gráfico) usando a linguagem JavaScript sem depender de plug-ins de navegador proprietários. Isso é possível graças ao advento da WebGL (Web Graphics Library), uma API em JavaScript disponível a partir do novo elemento canvas do HTML5. Ela pode renderizar usando Canvas, SVG e WebGL.
